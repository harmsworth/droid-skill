# 用户习惯规则

## 1. 核心原则

- **删前保后**：重复内容删除前面的，保留后面的
- **整句删除**：不要只删异常的几个字，要删整句或整段
- **范围整段删除**：从 startIdx 到 endIdx 之间的所有元素全部加入

## 2. 语气词检测

常见语气词：嗯、啊、呃、那个、就是、然后

- 孤立语气词（前后有内容）：标记但不自动删除
- 连续语气词（嗯啊、啊呃）：考虑删除
- 句首语气词+停顿：可能删除

## 3. 静音段处理

- >0.3s 的静音：自动预选删除
- 0.2-0.3s 的静音：根据上下文判断
- <0.2s 的静音：保留（自然停顿）

## 4. 重复句检测

- 相邻句子开头≥5字相同 → 删较短的整句
- 中间有残句的隔句重复 → 删前句+残句
- 部分重复（"我意思是…我意思是…"）→ 删前面部分

## 5. 卡顿词

- "那个那个" → 删第一个"那个"
- "就是就是" → 删第一个"就是"
- "然后然后" → 删第一个"然后"

## 6. 句内重复检测

模式：A + 中间内容 + A

例："好我们接下来好我们接下来做" → 删第一个"好我们接下来"

## 7. 连续语气词

- "嗯啊" → 标记
- "啊呃" → 标记
- "嗯嗯嗯" → 考虑删除后两个

## 8. 重说纠正

- 否定纠正："不是…是…" → 删"不是…"
- 部分重复纠正："我意思是…我是说…" → 删前面部分

## 9. 残句检测

- 话说一半 + 静音 → 整个残句删除
- 无意义片段（"这个这个…"）→ 删除
